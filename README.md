**Project Overview**

This project focuses on collecting and analyzing workforce, salary, ratings, jobs, and industry data for the top 500 companies listed on AmbitionBox.
It covers:

- Web scraping using Python

- Data cleaning and preprocessing

- Exploratory Data Analysis (EDA)

- Industry, salary, and location insights

- Correlation and multivariate analysis

- Visualizations using Python libraries

The final dataset provides valuable insights into company performance, salary trends, and employee satisfaction across industries.
